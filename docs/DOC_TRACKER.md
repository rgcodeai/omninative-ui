# Project Structure

```markdown
└── 📁 omninative-ui/
    ├── 📁 omninative_ui/          # Python Package: Base UI components library emulating OmniNative
    │   ├── 📄 __init__.py         # Re-exports the complete public API (backward compatibility)
    │   ├── 📄 tokens.py           # Central OMNINATIVE dictionary and design constants
    │   ├── 📄 icons.py            # Cached vector icons library
    │   ├── 📄 core.py             # Fundamental components (OWindow, OButton, OComboBox, etc.)
    │   ├── 📄 inputs.py           # Inputs and sliders (OLineEdit, OSpinBox, OSlider, etc.)
    │   ├── 📄 containers.py       # Containers and lists (OScrollArea, OTreeWidget, OVirtualTable)
    │   ├── 📄 media.py            # Multimedia components (OAudioPlayer, OImageViewer)
    │   ├── 📄 chat.py             # Chat interface components for AI (OChatView, OChatInput)
    │   └── 📄 _utils.py           # Internal utility functions
    ├── 📁 omninative_ui.egg-info/ # Package metadata generated by setuptools
    │   ├── 📄 dependency_links.txt# List of dependency links
    │   ├── 📄 PKG-INFO            # Package metadata
    │   ├── 📄 requires.txt        # Required dependencies
    │   ├── 📄 SOURCES.txt         # List of source files
    │   └── 📄 top_level.txt       # Top-level module name
    ├── 📁 examples/               # Package usage examples
    │   └── 📄 demo.py             # Demonstration interface exposing all components
    ├── 📄 pyproject.toml          # Packaging configuration (setuptools)
    ├── 📄 README.md               # Package documentation and installation instructions
    ├── 📄 LICENSE                 # LGPL-3.0 License
    ├── 📄 setup_env.bat           # Script to create and configure the Python virtual environment
    └── 📁 docs/
        ├── 📄 INDEX.md            # Main context router for documentation
        ├── 📄 DOC_TRACKER.md      # Project structure map in tree format
        ├── 📄 AI_PROMPT.md        # AI Instruction Manual for building UI
        └── 📁 core/
            ├── 📄 ui-system.md    # Central router and UI guide oriented to AIs
            ├── 📄 component-development.md # Guide for creating new UI components
            ├── 📄 ui-core.md      # API Reference for OWindow, OGroup, OButton, OComboBox, etc.
            ├── 📄 ui-inputs.md    # API Reference for OLineEdit, OTextBox, OSpinBox, OSlider
            ├── 📄 ui-containers.md# API Reference for OScrollArea, OTreeWidget, OTabs, OVirtualTable
            ├── 📄 ui-media.md     # API Reference for OAudioPlayer, OImageViewer
            └── 📄 ui-chat.md      # API Reference for OChatView, OChatInput
```
